## Descrição
Aplicação Web em ReactJS e NodeJS (REST API + Express) para analisar arquivos de texto com a LLM da OpenAI. 

## Funcionalidades
* Conversão de Arquivos de Texto.
* Analisador de Texto com o Chat GPT.
